COCO SOLO






Fecha de inauguración: c. 1920s.
Nombre original: Reserva Naval de Coco Solo
Uso anterior: Sirvió como una base para submarinos y, durante la Segunda Guerra Mundial, funcionó como estación aeronaval. Finalizado este conflicto, el uso militar de Coco Solo disminuyó y fue destinado para uso civil.

Fecha de reversión: 1999.
Nombre actual: Coco Solo
Uso actual: Esta área fue dividida en dos partes, estableciéndose en cada una un puerto de contenedores: Colon Container Terminal (CCT), en el norte; y Manzanillo International Terminal (MIT), en el sur.
